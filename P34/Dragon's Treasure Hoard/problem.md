## Title

Dragon's Treasure Hoard

## Slug

dragons-treasure-hoard

## Difficulty

Medium

## Description

A dragon has $n$ piles of gold and $n$ piles of gems, each ranked by size from 1 to $n$. The dragon values a combination of gold pile $i$ and gem pile $j$ as $i \times j$.The dragon is reorganizing its hoard and wants to find the median treasure value of all $n^2$ possible combinations. Given that $n$ is an odd number, calculate this median.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
