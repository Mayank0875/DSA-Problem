## Title
Solar Flare Spectrum

## Slug
solar-flare-spectrum

## Difficulty
Medium

## Description
A spectrometer records the dominant wavelength class of solar flares over time. Find the longest period where the sun emitted a unique class of flare each time without repetition.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
