## Title
Traffic Cone Zone

## Slug
traffic-cone-zone

## Difficulty
Easy

## Description
Road workers are marking off a square danger zone using traffic cones placed in a grid pattern. Given the total number of cones available in the truck, calculate the maximum number of cones that can form one side of the zone.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of cones available.

## Output Format
- Return single integer — the maximum possible side length r of the square zone that can be built using those cones.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
