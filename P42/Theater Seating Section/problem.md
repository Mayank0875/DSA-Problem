## Title
Theater Seating Section

## Slug
theater-seating-section

## Difficulty
Easy

## Description
An event organizer is setting up chairs for an outdoor performance. The VIP section must be a perfect square of chairs. You are given the total number of premium chairs available. Determine the maximum number of rows (which equals the number of columns) for this VIP square section.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of chairs available.

## Output Format
- Return single integer — the maximum possible side length r of the square section that can be built using those chairs.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
