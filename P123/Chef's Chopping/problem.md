## Title
Chef's Chopping

## Slug
chefs-chopping

## Difficulty
Medium

## Description
A chef chops vegetables. They can do a quick slice (1 piece) or a power chop (2 pieces). Power chops dull the knife momentarily, preventing consecutive power chops.

The chef starts at veggie 0 and wants to reach veggie $n$.

On each move, The chef can chop forward either **1 piece** (+1) or **2 pieces** (+2). However, precision is lost. Therefore, The chef **cannot perform two consecutive +2 power chops**.

Your task is to calculate the number of distinct valid ways The chef can reach veggie $n$. Since the answer can be very large, return it modulo $10^9 + 7$.

## Examples

### 1

#### Input
3

#### Output
3

#### Explanation
The valid paths to reach veggie 3 are:
1. 0 -> 1 -> 2 -> 3 (+1, +1, +1)
2. 0 -> 1 -> 3 (+1, +2)
3. 0 -> 2 -> 3 (+2, +1)

Note: The path starting with +2 is valid as long as the next jump is not +2.

### 2

#### Input
4

#### Output
4

#### Explanation
Valid paths:
1. 0 -> 1 -> 2 -> 3 -> 4 (+1, +1, +1, +1)
2. 0 -> 1 -> 2 -> 4 (+1, +1, +2)
3. 0 -> 1 -> 3 -> 4 (+1, +2, +1)
4. 0 -> 2 -> 3 -> 4 (+2, +1, +1)

Invalid path:
0 -> 2 -> 4 (+2, +2) is not allowed.

## Input Format
- The only input line has an integer $n$ — the target piece index.

## Output Format
- Return a single integer: the number of ways modulo $10^9 + 7$.

## Constraints
- 1 ≤ n ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB
