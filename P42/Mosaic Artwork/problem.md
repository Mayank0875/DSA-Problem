## Title
Mosaic Artwork

## Slug
mosaic-artwork

## Difficulty
Easy

## Description
An artist is creating a large floor mosaic using colored stone tiles. She wants the final piece to be a perfect square. You are given the total count of stone tiles she has collected. Calculate the maximum side length of the square mosaic she can complete using her current supply.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of stone tiles available.

## Output Format
- Return single integer — the maximum possible side length r of the square mosaic that can be built using those stone tiles.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
