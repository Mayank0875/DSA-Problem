## Title
Sugar Cube Pyramid

## Slug
sugar-cube-pyramid

## Difficulty
Easy

## Description
You are building a pyramid out of sugar cubes and need to construct the square base layer first. Given the total number of sugar cubes you have, determine the width of the largest square base you can build.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of cubes available.

## Output Format
- Return single integer — the maximum possible side length r of the square base that can be built using those cubes.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
