## Title
Textile Pattern Loom

## Slug
textile-pattern-loom

## Difficulty
Medium

## Description
A loom weaves a pattern of colored threads (characters). The designer wants to find the widest strip of the fabric where every thread color is unique. Calculate the width (number of threads) of this unique strip.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
