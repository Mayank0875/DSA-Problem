## Title

Fountain Installation

## Slug

fountain-installation

## Difficulty

Medium

## Description

A park designer is placing `n` large fountains in an `n x n` garden. For aesthetic reasons, the designer wants to ensure that no two fountains are in the same row, column, or diagonal line of sight. How many different valid designs are possible for the `n` fountains?
## Examples

### 1

#### Input

1

#### Output

1

#### Explanation

On a 1x1 board, there is only one square, and placing one queen there is a valid solution.
    
### 2

#### Input

4

#### Output

2

#### Explanation
There are two distinct ways to place 4 queens on a 4x4 board safely.

## Input Format  

- The input consists of a single line containing one integer, `n`.

## Output Format  

- Return single integer representing the total number of distinct solutions.
  

## Constraints  

- 1 ≤ n ≤ 8

## Time Limit

2 second

## Memory Limit

256 MB

## Tags

backtracking, recursion
