## Title

Factory Widget Production

## Slug

factory-widget-production

## Difficulty

Medium

## Description

A new factory has $n$ production lines and operates for $n$ shifts. It is known that in shift $i$, production line $j$ manufactures exactly $i \times j$ widgets. The factory manager needs to analyze the production data from all $n \times n$ possible line-shift combinations.Your task is to find the median production value (the middle element) from the set of all $n^2$ production values. It is guaranteed that $n$ is an odd number.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
  

## Input Format  

- The only input line has an integer n.

## Output Format  

- Return one integer: the answer to the task.
