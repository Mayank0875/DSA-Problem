## Title
Chocolate Box Layer

## Slug
chocolate-box-layer

## Difficulty
Easy

## Description
A chocolatier is designing a premium box where the chocolates are arranged in a single square layer. You are given the total number of chocolates produced in a batch. Determine the side length of the largest square box that can be filled completely with this batch.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of chocolates available.

## Output Format
- Return single integer — the maximum possible side length r of the square box layer that can be built using those chocolates.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
