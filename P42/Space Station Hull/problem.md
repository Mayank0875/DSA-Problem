## Title
Space Station Hull

## Slug
space-station-hull

## Difficulty
Easy

## Description
Engineers are welding together hull plates to form a large square section of a space station wall. You are given the total count of hull plates available. Determine the side dimension of the largest square section that can be completed.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of hull plates available.

## Output Format
- Return single integer — the maximum possible side length r of the square section that can be built using those hull plates.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
