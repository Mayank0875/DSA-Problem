## Title
Medal Tiers

## Slug
medal-tiers

## Difficulty
Easy

## Description
A game awards $n$ medals to a player. There are 5 tiers: Iron, Bronze, Silver, Gold, Platinum. The medals are displayed in non-decreasing order of value. How many display cases are possible?

## Examples

### 1

#### Input
1

#### Output
5

#### Explanation
There are 5 valid sequences of length 1.

### 2

#### Input
2

#### Output
15

#### Explanation
There are 15 valid sorted sequences of length 2 (e.g., aa, ae, ai, ao, au, ee, etc.).

## Input Format
- The input consists of a single integer n, representing the required length/quantity.

## Output Format
- Return a single integer representing the total number of valid sorted sequences.

## Constraints
- 1 ≤ n ≤ 50

## Time Limit
2 second

## Memory Limit
256 MB

dynamic-programming, math
