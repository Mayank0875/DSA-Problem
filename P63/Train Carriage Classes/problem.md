## Title
Train Carriage Classes

## Slug
train-carriage-classes

## Difficulty
Easy

## Description
A train has $n$ carriages. There are 5 classes of service. To streamline service, carriages are arranged in non-decreasing order of class luxury. How many different train configurations are possible?

## Examples

### 1

#### Input
1

#### Output
5

#### Explanation
There are 5 valid sequences of length 1.

### 2

#### Input
2

#### Output
15

#### Explanation
There are 15 valid sorted sequences of length 2 (e.g., aa, ae, ai, ao, au, ee, etc.).

## Input Format
- The input consists of a single integer n, representing the required length/quantity.

## Output Format
- Return a single integer representing the total number of valid sorted sequences.

## Constraints
- 1 ≤ n ≤ 50

## Time Limit
2 second

## Memory Limit
256 MB
