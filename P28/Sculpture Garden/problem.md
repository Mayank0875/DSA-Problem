## Title

Sculpture Garden

## Slug

sculpture-garden

## Difficulty

Medium

## Description

You are designing a sculpture garden on an `n x n` plot of land. You must place `n` large sculptures. To ensure visitors can see each one clearly from all angles, no two sculptures can be aligned horizontally, vertically, or diagonally. How many distinct garden layouts are possible?
## Examples

### 1

#### Input

1

#### Output

1

#### Explanation

On a 1x1 board, there is only one square, and placing one queen there is a valid solution.
    
### 2

#### Input

4

#### Output

2

#### Explanation
There are two distinct ways to place 4 queens on a 4x4 board safely.

## Input Format  

- The input consists of a single line containing one integer, `n`.

## Output Format  

- Return single integer representing the total number of distinct solutions.
  

## Constraints  

- 1 ≤ n ≤ 8

## Time Limit

2 second

## Memory Limit

256 MB

## Tags

backtracking, recursion
