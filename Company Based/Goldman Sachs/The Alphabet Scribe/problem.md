## Title

The Alphabet Scribe

## Slug

alphabet-scribe-check

## Difficulty

Easy

## Description

You are an apprentice to a master scribe who is testing a new set of magical quills. To ensure a quill is fully functional, the scribe writes a test sentence. A successful test sentence is known as a "pangram," which is a sentence that contains every single letter of the English alphabet at least once.

You are given the test string sentence, which consists only of lowercase English letters. Your task is to check the scribe's work and determine if the sentence is a valid pangram.

## Examples

### 1

#### Input

thequickbrownfoxjumpsoverthelazydog

#### Output

Yes

#### Explanation

The input sentence contains at least one of every letter of the English alphabet.
    
### 2

#### Input

cipherschools

#### Output

No

#### Explanation

The input sentence dosn't contains at least one of every letter of the English alphabet.
  

## Input Format  

- The input consists of a single line containing the string sentence.

## Output Format  

- Return yes if the sentence is a pangram Otherwise no.
  

## Constraints  

- 1 ≤ sentence.length ≤ 1000
- sentence consists of lowercase English letters.

## Time Limit

2 second

## Memory Limit

256 MB

## Tags

string, array
