## Title
Forest Tree Line

## Slug
forest-tree-line

## Difficulty
Medium

## Description
A transect line through a forest records tree species. Ecologists want to find the longest stretch of the line where every tree encountered is of a different species. Calculate the number of trees in this stretch.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
