## Title
Sound Volume Levels

## Slug
sound-volume-levels

## Difficulty
Easy

## Description
A volume test plays $n$ tones. The volume is set to one of 5 levels. The test requires the volume to never decrease. How many test sequences can be run?

## Examples

### 1

#### Input
1

#### Output
5

#### Explanation
There are 5 valid sequences of length 1.

### 2

#### Input
2

#### Output
15

#### Explanation
There are 15 valid sorted sequences of length 2 (e.g., aa, ae, ai, ao, au, ee, etc.).

## Input Format
- The input consists of a single integer n, representing the required length/quantity.

## Output Format
- Return a single integer representing the total number of valid sorted sequences.

## Constraints
- 1 ≤ n ≤ 50

## Time Limit
2 second

## Memory Limit
256 MB

dynamic-programming, math
