## Title

Deck Pair Finder

## Slug

deck-pair-finder

## Difficulty

Easy

## Description

You are given a list of cards in a playing deck, represented by their integer values. In this game, a 'pair' is a card value that appears exactly twice in the hand. Given the list of card values, identify all pairs. Return them in increasing order. If no pairs exist, return an empty list.

## Examples

### 1
#### Input
8
1 2 2 3 3 3 4 4

#### Output
2 4

#### Explanation
2 appears twice, 4 appears twice. 3 appears thrice (not a pair), 1 appears once.

### 2
#### Input
5
5 5 5 6 7

#### Output


#### Explanation
No number appears exactly twice.

## Input Format
- First line: integer n — number of elements.
- Second line: n space-separated integers.

## Output Format
- Return all integers that appear exactly twice in increasing order, as a list/array. If none, return an empty list/array.

## Constraints
- 1 ≤ n ≤ 10^5
- Values fit in 32-bit signed integer

## Time Limit
1 second

## Memory Limit
256 MB
