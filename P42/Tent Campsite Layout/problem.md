## Title
Tent Campsite Layout

## Slug
tent-campsite-layout

## Difficulty
Easy

## Description
A scout leader wants to set up tents in a square grid formation for inspection. You are given the total number of tents the troop has. Determine the maximum number of tents that can form one side of this square grid.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of tents available.

## Output Format
- Return single integer — the maximum possible side length r of the square grid that can be built using those tents.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
