## Title
Train Cargo Variety

## Slug
train-cargo-variety

## Difficulty
Medium

## Description
A freight train passes by, carrying different types of cargo containers (chars). A logistics inspector notes the longest continuous section of the train where every container holds a different type of cargo. Calculate this length.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
