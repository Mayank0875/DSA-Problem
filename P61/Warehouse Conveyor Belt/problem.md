## Title
Warehouse Conveyor Belt

## Slug
warehouse-conveyor-belt

## Difficulty
Medium

## Description
Packages with destination codes pass on a belt. The diverter system works best when processing unique destinations. Find the longest sequence of packages where every destination code is distinct.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
