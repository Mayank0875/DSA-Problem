## Title
Stamp Collection Page

## Slug
stamp-collection-page

## Difficulty
Easy

## Description
A philatelist wants to arrange stamps on a large display page in a square formation. You are given the total number of unique stamps in the collection. Your task is to find the maximum number of stamps that can fit along the edge of this square arrangement.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of stamps available.

## Output Format
- Return single integer — the maximum possible side length r of the square display that can be built using those stamps.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
