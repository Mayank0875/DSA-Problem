## Title

Stock Profit Matrix

## Slug

stock-profit-matrix

## Difficulty

Medium

## Description

An analyst is studying $n$ different stocks over $n$ consecutive days. The projected profit for stock $i$ (ranked 1 to $n$) on day $j$ (ranked 1 to $n$) is $i \times j$ dollars.The analyst needs to find the median profit value from the $n^2$ entries in this projection matrix to write a market summary. Given that $n$ is an odd integer, find the median.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
