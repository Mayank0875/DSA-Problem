## Title
Gemstone Necklace Variety

## Slug
gemstone-necklace-variety

## Difficulty
Medium

## Description
A jeweler is examining a long necklace made of various types of gemstones, each represented by a character code. They want to cut a segment of the necklace to sell as a premium bracelet. The premium bracelet must consist of a continuous run of gems where every gem is unique. Find the max length of such a bracelet.

## Examples

### 1

#### Input
abcabcbb

#### Output
3

#### Explanation
The answer is "abc", with the length of 3.

### 2

#### Input
bbbbb

#### Output
1

#### Explanation
The answer is "b", with the length of 1.

## Input Format
- The input consists of a single line containing the string `s`.

## Output Format
- Return a single integer representing the maximum length of a substring with unique characters.

## Constraints
- 1 ≤ length(s) ≤ 10^5

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
string, hash-table, sliding-window
