## Title
Spice Level Menu

## Slug
spice-level-menu

## Difficulty
Easy

## Description
A restaurant offers a tasting menu of $n$ dishes. Each dish has a spice level from 1 to 5. The chef arranges the menu so that the spice level never decreases. How many tasting menu progressions are there?

## Examples

### 1

#### Input
1

#### Output
5

#### Explanation
There are 5 valid sequences of length 1.

### 2

#### Input
2

#### Output
15

#### Explanation
There are 15 valid sorted sequences of length 2 (e.g., aa, ae, ai, ao, au, ee, etc.).

## Input Format
- The input consists of a single integer n, representing the required length/quantity.

## Output Format
- Return a single integer representing the total number of valid sorted sequences.

## Constraints
- 1 ≤ n ≤ 50

## Time Limit
2 second

## Memory Limit
256 MB

dynamic-programming, math
