## Title
Musical Scale Practice

## Slug
musical-scale-practice

## Difficulty
Easy

## Description
A musician practices a pentatonic scale (5 notes: Do, Re, Mi, Sol, La). They play a sequence of $n$ notes. To practice ascending runs, every note played must be the same or higher than the previous one. Count the number of valid ascending practice sequences.

## Examples

### 1

#### Input
1

#### Output
5

#### Explanation
There are 5 valid sequences of length 1.

### 2

#### Input
2

#### Output
15

#### Explanation
There are 15 valid sorted sequences of length 2 (e.g., aa, ae, ai, ao, au, ee, etc.).

## Input Format
- The input consists of a single integer n, representing the required length/quantity.

## Output Format
- Return a single integer representing the total number of valid sorted sequences.

## Constraints
- 1 ≤ n ≤ 50

## Time Limit
2 second

## Memory Limit
256 MB
