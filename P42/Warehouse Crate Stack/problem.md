## Title
Warehouse Crate Stack

## Slug
warehouse-crate-stack

## Difficulty
Easy

## Description
A warehouse manager needs to arrange crates on the floor in a single layer. To ensure stability and easy access, the crates must form a perfect square area. You are provided with the total number of crates available. Your objective is to find the maximum number of crates that can form the side of this square arrangement.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of crates available.

## Output Format
- Return single integer — the maximum possible side length r of the square arrangement that can be built using those crates.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
