## Title

Board Game Points

## Slug

board-game-points

## Difficulty

Medium

## Description

A new board game is played on an $n \times n$ grid (where $n$ is odd). The rows and columns are indexed from 1 to $n$. Landing on the square at row $i$ and column $j$ gives the player $i \times j$ points.To assess the fairness of the board, the designer needs to know the median point value of all $n^2$ squares.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
