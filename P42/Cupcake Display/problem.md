## Title
Cupcake Display

## Slug
cupcake-display

## Difficulty
Easy

## Description
A baker wants to arrange cupcakes on a large square table for a wedding. The arrangement must be a perfect square. Given the total number of cupcakes baked, find the number of cupcakes along the edge of the largest possible square arrangement.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of cupcakes available.

## Output Format
- Return single integer — the maximum possible side length r of the square arrangement that can be built using those cupcakes.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
