## Title
Window Pane Lattice

## Slug
window-pane-lattice

## Difficulty
Easy

## Description
A glazier is restoring an old window that uses a grid of small square glass panes. The window frame must be square. Given the total number of glass panes available, determine the maximum number of panes along one side of the window.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of panes available.

## Output Format
- Return single integer — the maximum possible side length r of the square window that can be built using those panes.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
