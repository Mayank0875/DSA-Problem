## Title

Seed and Fertilizer Pairs

## Slug

seed-fertilizer-pairs

## Difficulty

Medium

## Description

A botanist is testing $n$ types of seeds and $n$ types of fertilizer, both ranked from 1 to $n$. When seed type $i$ is combined with fertilizer type $j$, the resulting plant produces $i \times j$ kilograms of fruit.The botanist needs to report the median production value from all $n^2$ combinations to a research journal. Given that $n$ is an odd number, find the median.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
