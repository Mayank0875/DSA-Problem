## Title
Library Bookshelf Area

## Slug
library-bookshelf-area

## Difficulty
Easy

## Description
A librarian is setting up a new reading zone with identical bookshelves arranged in a square formation. You are given the total number of bookshelves available. Your goal is to determine the number of shelves along one side of the largest possible square formation.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of bookshelves available.

## Output Format
- Return single integer — the maximum possible side length r of the square formation that can be built using those bookshelves.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
