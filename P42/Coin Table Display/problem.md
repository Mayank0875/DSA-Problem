## Title
Coin Table Display

## Slug
coin-table-display

## Difficulty
Easy

## Description
A collector wants to display rare coins on a table in a solid square formation. You are given the total number of coins in the collection. Find the number of coins along the edge of the largest possible square formation.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of coins available.

## Output Format
- Return single integer — the maximum possible side length r of the square formation that can be built using those coins.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
