## Title

Ancient Scroll Reversal

## Slug

ancient-scroll-reversal

## Difficulty

Easy

## Description

Your challenge is to write a function that reverses this string. Because the scroll's data is very large and your field computer has limited resources, you cannot simply create a new string. You must reverse the string `in-place`, by modifying the given string (or character array) directly.

## Examples

### 1

#### Input

hello

#### Output

olleh

#### Explanation

The input string `hello` is modified in-place to become `olleh`.
    
### 2

#### Input

level

#### Output

level

#### Explanation

The input string `level` is modified in-place to become `level`.
  

## Input Format  

- The only input line has an string s.

## Output Format  

- Return a reverse string without using space.
  

## Constraints  

- 3 ≤ s.length ≤ 2000

## Time Limit

2 second

## Memory Limit

256 MB

## Tags

string, two-pointers, in-place
