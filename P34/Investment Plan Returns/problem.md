## Title

Investment Plan Returns

## Slug

table-multiplication

## Difficulty

Medium

## Description

An investor is considering $n$ different investment plans and $n$ different risk levels, both categorized from 1 to $n$. The estimated return for plan $i$ at risk level $j$ is $i \times j$ units.The investor wants to know the median return value from all $n^2$ possible combinations to gauge the typical outcome. Given that $n$ is an odd integer, calculate this value.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The numbers in increasing order are [1,2,2,3,3,4,6,6,9], so the answer is 3. 
    
### 2

#### Input

5

#### Output

8

#### Explanation
The middle element is 8.
  

## Input Format  

- The only input line has an integer n.

## Output Format  

- Return one integer: the answer to the task.
