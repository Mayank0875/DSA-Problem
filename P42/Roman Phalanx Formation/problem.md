## Title
Roman Phalanx Formation

## Slug
roman-phalanx-formation

## Difficulty
Easy

## Description
A Roman Centurion is preparing his troops for battle. He wants to arrange his soldiers into a solid square formation to maximize defense. You are given the total count of soldiers available. Your task is to determine the maximum number of soldiers that can stand on one side of the largest possible square formation. Any extra soldiers who cannot fit into the perfect square will be placed in reserve.

## Examples

### 1

#### Input
10

#### Output
3

#### Explanation
3² = 9 ≤ 10, 4² = 16 > 10

### 2

#### Input
25

#### Output
5

#### Explanation
5² = 25 ≤ 25, 6² = 36 > 25

## Input Format
- A single integer x — the total number of soldiers available.

## Output Format
- Return single integer — the maximum possible side length r of the square formation that can be built using those soldiers.

## Constraints
- 0 ≤ x ≤ 1e9

## Time Limit
2 second

## Memory Limit
256 MB

## Tags
binary-search, math, number-theory
