## Title

Matrix Median

## Slug

matrix-median

## Difficulty

Medium

## Description

Given an integer n, consider an n x n multiplication table. Find the median value when all numbers in the table are sorted in non-decreasing order. Note that n is odd.

For example, the 3x3 multiplication table is:
    [[1, 2, 3], [2, 4, 6], [3, 6, 9]]
The sorted list is [1,2,2,3,3,4,6,6,9]. The median (the 5th element in a 1-indexed list of 9 elements) is 3.

## Examples

### 1

#### Input

3

#### Output

3

#### Explanation

The sorted list is [1,2,2,3,3,4,6,6,9]. The median (5th element) is 3.

### 2

#### Input

5

#### Output

8

#### Explanation

The sorted list has 25 elements. The median (13th element) is 8.

## Input Format

- One integer n.

## Output Format

- One integer: the median.

## Constraints

- 1 ≤ n ≤ 1e6

## Time Limit

2 seconds

## Memory Limit

256 MB
